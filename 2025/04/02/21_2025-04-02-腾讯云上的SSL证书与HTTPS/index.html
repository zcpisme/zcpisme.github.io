改进的海龟交易策略

经过分析发现，原版的交易策略里，唐奇安通道在空仓时表现极差，所以改用突破布林带上轨时，意味着市场可能正在经历超买，所以开空单，其余保持不变。

# region imports
from AlgorithmImports import *
from datetime import timedelta
# endregion
class TurtleTrading(QCAlgorithm):
    def initialize(self):
        self.set_start_date(2024, 1, 1)
        self.set_end_date(2025, 1, 1)
        self.set_cash(10000000)
        self.SetBrokerageModel(BrokerageName.InteractiveBrokersBrokerage, AccountType.Margin)

        self.nasdaq = self.add_equity('AAPL', Resolution.DAILY, data_normalization_mode=DataNormalizationMode.RAW).symbol


        self.entry_length = 20
        self.exit_length = 10
        self.atr_length = 14
        self.risk_percent = 0.01
        self.max_position = 10
        self.donchian_entry = self.dch(self.nasdaq, self.entry_length, self.entry_length)
        self.donchian_exit = self.dch(self.nasdaq, self.exit_length, self.exit_length)
        self._bb = self.bb(self.nasdaq, 20, 1.6)

        self.long_entry = float('inf')
        self.short_entry = 0
        self.long_exit = 0
        self.short_exit = float('inf')

        # self.set_warm_up(50)

        self._atr = self.atr(self.nasdaq, self.atr_length, MovingAverageType.Simple, Resolution.DAILY)

        self.entry_price = 0
        self.position_count = 0
        self.stop_loss = 0
        self.consolidate(self.nasdaq, timedelta(days=1), self.on_consolidated)

    def on_consolidated(self, bar: TradeBar):
        # Plot the candlestick data
        self.plot("DonchianChannel", "AAPL", bar)

    def on_data(self, data: Slice):
        if not (self.donchian_entry.is_ready and self.donchian_exit.is_ready and self._atr.is_ready):
            return

        current_price = self.securities[self.nasdaq].price


        if self.donchian_entry.is_ready:
            # The current value of self._dch is represented by self._dch.current.value
            self.plot("DonchianChannel", "dch", self.donchian_entry.current.value)
            # Plot all attributes of self._dch
            self.plot("DonchianChannel", "upper_band", self.donchian_entry.upper_band.current.value)
            self.plot("DonchianChannel", "lower_band", self.donchian_entry.lower_band.current.value)


        if self._atr.is_ready:
            # The current value of self._atr is represented by self._atr.current.value
            self.plot("AverageTrueRange", "atr", self._atr.current.value)


        holdings = self.portfolio[self.nasdaq].quantity
        dollar_risk = self.portfolio.total_portfolio_value * self.risk_percent
        atr_value = self._atr.current.value
        contract_value = current_price
        position_size = dollar_risk / (atr_value * contract_value)
        position_size = dollar_risk / (atr_value)

        if holdings == 0 and current_price > self.long_entry:
            self.debug('long')
            self.entry_price = current_price
            self.stop_loss = current_price - 2 * atr_value
            self.position_count = 1
            self.market_order(self.nasdaq, position_size)
        elif holdings == 0 and current_price >= self.short_entry:
            self.debug('short')
            self.entry_price = current_price
            self.stop_loss = current_price + 2 * atr_value
            self.position_count = 1
            self.market_order(self.nasdaq, -position_size)
        elif abs(holdings) < self.max_position * position_size:
            if (holdings > 0 and current_price > self.entry_price + 0.5 * atr_value) or \
               (holdings < 0 and current_price < self.entry_price - 0.5 * atr_value):
                self.position_count += 1
                self.entry_price = current_price
                self.market_order(self.nasdaq, position_size if holdings > 0 else -position_size)
        if (holdings > 0 and current_price < self.stop_loss) or \
           (holdings < 0 and current_price > self.stop_loss):
            self.liquidate(self.nasdaq)
            self.position_count = 0
        if holdings > 0 and current_price < self.long_exit:
            self.liquidate(self.nasdaq)
        elif holdings < 0 and current_price <= self.short_exit:
            self.liquidate(self.nasdaq)

        self.long_entry = self.donchian_entry.upper_band.current.value
        self.short_entry = self.donchian_entry.lower_band.current.value
        self.short_entry = self._bb.upper_band.current.value
        self.long_exit = self.donchian_exit.lower_band.current.value
        self.short_exit = self.donchian_exit.upper_band.current.value
        self.short_exit = self._bb.lower_band.current.value

上图可以看到，在AAPL上回测，2024带来了46%的收益率，超过基准（从年初一直持有股票到年末，35%）足足有10%。

Deepseek详细策略回测报告：

一、回测概况

  • 回测时间：2024年1月1日 - 2025年1月1日

  • 初始资金：$10,000,000

  • 最终资金：$14,671,111.10（增长46.71%） 年化收益率：46.61%（按复利计算） 交易次数：33次（中等频率，约每周交易1次）

二、风险与稳定性

  • 最大回撤：16.5%（历史上资金最大下跌幅度，较高风险）
  • 资金波动率：20%（收益波动较大，属于高风险策略）
  • 夏普比率：1.37（数值超过1，说明收益稳定性优秀）
  • 索提诺比率：1.345（考虑亏损风险后，稳定性仍较好）

三、市场表现对比

  • 相对市场的超额收益（Alpha）：0（收益与市场同步，未跑赢或跑输大盘）
  • 市场相关性（Beta）：0（策略表现与市场波动完全无关）
  • 信息比率：1.645（策略跑赢市场基准，表现优于同类）

四、交易成本与适用性

  • 总交易费用：$4,415.68（成本较低，仅占收益的0.03%）
  • 换手率：3.97%（交易频率较低，适合长期持仓）
  • 适合资金规模：$3.1亿美元（适用于大资金量操作）
  • 主要交易资产：AAPL相关金融产品（代码AAPLR735QTJ8XC9X）

这个策略收益非常突出（46.71%增长，年化46.61%），但风险也较高：
优点：

  1. 高胜率（80%交易盈利），平均盈利是亏损的4.4倍；
  2. 与市场无关（Beta=0），适合分散投资风险；
  3. 交易成本极低，适合大资金操作（3.1亿美元容量）。

缺点：

  1. 最大亏损达16.5%，短期可能承受较大波动；
  2. 收益完全依赖策略本身（Alpha=0），若策略失效可能失去盈利来源；
  3. 与市场零相关性（Beta=0）既是优点也是缺点，无法享受市场整体上涨红利。

适用场景：适合风险承受能力较强、需要配置与市场无关策略的大型资金，但需警惕高波动风险。

最后

换一个例子，在百度BIDU这种年年跌的股票上，2024全年跌幅超过24%，使用上述策略能将跌幅控制在3%左右。

海龟交易策略还是很强大的，这还只是简单改进了一个点，后续加入更多分析后，应该能进一步提升指标，就留给未来大家自己发掘了。
